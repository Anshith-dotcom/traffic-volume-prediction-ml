# Traffic Volume Prediction using Machine Learning

## Problem Statement
Traffic congestion is a major challenge in urban areas. This project predicts
traffic volume using temporal and environmental features with machine learning.

## Dataset
A structured dataset containing:
- Hour of day
- Day of week
- Holiday indicator
- Temperature
- Rainfall

## Methodology
- Data preprocessing and feature scaling
- Training regression-based ML models
- Model evaluation using MAE and RMSE

## Models Used
- Linear Regression
- Random Forest Regressor

## Results
Random Forest performed better than Linear Regression,
capturing non-linear traffic patterns effectively.

## Conclusion
The project demonstrates the application of classical
machine learning models for traffic volume prediction.
